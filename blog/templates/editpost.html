Formatting Basics

When creating a blog post, you can use either markdown or html to help with formatting and customization.


Headers:

# Largest Header (h1)
## Smaller Header (h2)
...
###### Smallest header (h6)


Emphasis:

Wrap text in asterisks for *italics*
Double wrap for **bold**


Code:

For inline code, wrap in `backticks`
For block code, wrap in <pre> tags
For example:

<pre>
def print_nums(num1, num2):
    for i in range(10):
        print(i)
</pre>

This will render as: 



def print_nums(num1, num2):
    for i in range(10):
        print(i)
